RHEL8安装教程

在 VMware 中安装 RHEL 8（Red Hat Enterprise Linux 8）系统的步骤如下。

1. 下载 RHEL 8 ISO 文件

  • 前往 Red Hat 官方网站，下载 RHEL 8 的 ISO 安装镜像。
  • 需要先创建一个 Red Hat 账户才能进行下载。如果你还没有 Red Hat 账户，注册一个免费的开发者账户即可获得免费的订阅权限用于个人学习和开发。

    或者用班群里RHEL8的压缩包（推荐）

2. 安装 VMware Workstation Player

如果你还没有安装 VMware Workstation Player 或 Pro，请先从 VMware 官网下载并安装：

  • 下载地址：VMware 官网。
  • 请看我的上一章VMware虚拟机安装按照提示进行安装，安装完成后启动 VMware。

3. 创建新的虚拟机

1. 打开 VMware 并创建新的虚拟机

  • 在 VMware Workstation Player 中，点击“创建新的虚拟机”。
  • 选择“使用光盘映像文件（ISO）”并选择之前下载的 RHEL 8 ISO 文件。

2. 选择操作系统

  • 在 VMware 提示的“客户机操作系统”选项中，选择“Linux”，然后从下拉菜单中选择“Red Hat Enterprise Linux 8”作为操作系统版本。如果没有 RHEL 8 的选项，可以选择“Red Hat Enterprise Linux 7”作为替代。

3. 命名虚拟机

  • 给你的虚拟机取一个合适的名称，例如“RHEL 8”。
  • 选择虚拟机文件的存储位置（你可以将其保存在你的硬盘中的任何文件夹下）。

4. 分配内存（RAM）

  • 为虚拟机分配内存，建议至少分配 2 GB（2048 MB） 或更多内存，具体可以根据你的硬件配置和 RHEL 8 的使用需求来分配。

5. 创建虚拟硬盘

  • 选择创建一个新的虚拟硬盘，推荐至少分配 20 GB 的存储空间，具体大小可以根据需要进行调整。
  • 选择“单一文件”或“多文件分配”，大多数情况下，单一文件更简单。

4. 自定义虚拟机设置（可选）

在虚拟机创建完成后，可以进入虚拟机的“设置”菜单，根据需求进行更多的自定义设置：

  • 处理器：可以增加虚拟机的 CPU 核心数，建议至少 2 个核心以获得更好的性能。
  • 网络：保持默认的 NAT 模式即可，这样虚拟机能够访问外网。

5. 启动虚拟机并安装 RHEL 8

1. 启动虚拟机

  • 创建完成后，选择刚创建的 RHEL 8 虚拟机，然后点击“启动虚拟机”。
  • 虚拟机会启动并加载你选择的 RHEL 8 ISO 文件，进入 Red Hat 安装程序。

2. 选择安装选项

  • 在启动后的第一个界面，选择“Install Red Hat Enterprise Linux 8.0”，按下 Enter 键开始安装。

3. 选择语言和键盘布局

  • 选择你想要使用的语言和键盘布局，然后点击“继续”。

4. 配置安装目标和软件选择

  • 安装目标：在“安装目标”部分，你需要选择虚拟硬盘作为安装位置。选择虚拟硬盘，并选择“自动分区”选项，这样系统会自动分区并格式化硬盘。
  • 软件选择：在“软件选择”部分，选择你想要安装的 RHEL 8 软件包。你可以选择“Server with GUI”（带有图形界面的服务器）或“Minimal Install”（最小化安装）。如果你想要图形界面，建议选择“Server with GUI”。

5. 设置网络和主机名

  • 在安装界面中，选择“网络和主机名”，为你的系统指定一个主机名（例如：rhel8.local），然后启用网络接口，这样系统可以在安装后直接连接到互联网。

6. 开始安装

  • 配置完成后，点击“开始安装”按钮开始安装 RHEL 8。
  • 在安装过程中，你需要设置 root 密码 并创建一个新的用户账户。确保密码设置复杂且安全。

6. 完成安装

  • 安装完成后，点击“重启”按钮。
  • 虚拟机会重启并进入你刚刚安装的 RHEL 8 系统。

7. 注册 Red Hat 订阅（可选）

  • 如果你打算使用 Red Hat 的官方软件包源和更新服务，你需要注册你的系统。可以通过运行以下命令来注册：

  • 系统会提示你输入 Red Hat 账户的用户名和密码。

  • 你可以通过 subscription-manager 工具来启用或禁用软件源和订阅。

8. 安装 VMware Tools（增强工具）

  • 进入系统后，安装 VMware Tools 来提升虚拟机的性能和兼容性。

  • 在 VMware 菜单栏中，选择“虚拟机” > “安装 VMware Tools”。

9. 完成设置
